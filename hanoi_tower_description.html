HanoiTower Description

The tower of Hanoi is a puzzle invented by E. Lucas in 1883.

The World provides 3 pegs (left, middle and right) and 5 disks.
Disks are arranged from largest on the bottoms to smallest on top.

The game begins in the following format:
  • Peg A - 1,2,3,4,5 (were number respresents ring number/size)
  • Peg B - empty
  • Peg C - empty

Objective of Game

The towers of Hanoi puzzle asks for the minimum number of moves required to move the stack from one rod to another.

Rules

  1. Moves are allowed only if they place smaller disks on top of larger disks.
  2. You are allowed a maximum of 50 moves.
  3. An invalid move will still count towards the overall score.

Actions

Action = int supplied in relation to following actions (0-5):

  • TAKE_LEFT_MOVE_MIDDLE
  • TAKE_MIDDLE_MOVE_LEFT
  • TAKE_RIGHT_MOVE_LEFT
  • TAKE_LEFT_MOVE_RIGHT
  • TAKE_MIDDLE_MOVE_RIGHT
  • TAKE_RIGHT_MOVE_MIDDLE

States

State = "s1,s2,s3" where each represents a the element on a stack where:

  • s1 = left stack.
  • s2 = middle stack.
  • s3 = right stack.

Score

Score = "s1,s2,s3" where:
  • s1 = moves taken to complete puzzle.
  • s2 = maximum amount of moves allowed.
  • s3 = overall which is determined by (s2 - s1).
